# Emotion-Recognition
🔍 Project: Emotion Recognition

📊 Description: Developed a comprehensive emotion recognition system using the Toronto Emotional Speech Set (TESS) dataset. 
The project involved:
1. Loading and preprocessing the dataset
2. Extracting features using MFCC
3. Building and training an LSTM model
4. Evaluating model performance with accuracy and confusion matrix

💻 Technologies Used: Python, Keras, Librosa, Seaborn, Matplotlib, Scikit-learn

🧠 Key Learnings:
1. Enhanced understanding of speech signal processing
2. Improved skills in deep learning model implementation
3. Hands-on experience with real-world data
